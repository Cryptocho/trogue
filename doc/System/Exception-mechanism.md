# 异常机制设计文档

## 眩晕 (Stun)

**效果**：使目标陷入眩晕状态，无法行动。

| 属性 | 数值 |
|------|------|
| 技能使用 | 禁止 |
| 控制抗性 | 减少 15% |
| 准度 (hitRate) | 减少 7 点 |
| 闪避 (dodge) | 减少 7 点 |

---

## 中毒 (Poison)

**效果**：每回合持续扣除生命值。

| 属性 | 数值 |
|------|------|
| 每回合伤害 | 生命上限 × 1% + 1（中毒基础伤害） |

> 公式：`tick_damage = max_hp * 0.01 + 1`

---

## 流血 (Bleed)

**效果**：每回合持续扣除生命值，伤害比中毒更高。

| 属性 | 数值 |
|------|------|
| 每回合伤害 | 生命上限 × 3% + 2（流血基础伤害） |

> 公式：`tick_damage = max_hp * 0.03 + 2`

---

## 击退 (Knockback)

**效果**：向攻击方向的反方向移动一格。

| 情况 | 结果 |
|------|------|
| 目标格为障碍物 | 取消移动，触发碰撞伤害 |
| 目标格为墙体 | 取消移动，触发碰撞伤害 |
| 目标格空旷 | 正常移动 |

**碰撞伤害公式**：

> `collision_damage = attack_damage × 0.25 × (1 + (strength - 10) × 0.02 + (agility - 10) × 0.01)`

---

## 虚弱 (Weaken)

**效果**：降低输出能力，增加受击伤害。

| 属性 | 数值 |
|------|------|
| 总伤害 | 减少 9% |
| 技能精力消耗 | 增加 9% |
| 受到伤害 | 增加 6% |
